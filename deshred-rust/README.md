# Consume decoded shredstream using rust

## Setup

- Install rust ecosystems using [rustup](https://rustup.rs)
```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

- Your favorite IDE

## Run
- Dev:
```sh
cargo run http://IP:PORT
```
- Release

```sh
cargo run --release http://IP:PORT
```
